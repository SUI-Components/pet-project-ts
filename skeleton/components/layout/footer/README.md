### Basic usage

#### Import package and use the component

```js
import LayoutFooter from '@adv-ui/sui-layout-footer'

return (<LayoutFooter />)
```

#### Import the styles (Sass)

```css
@import '~@s-ui/theme/lib/index';
/* @import 'your theme'; */
@import '~@adv-ui/sui-layout-footer/lib/index';
```
